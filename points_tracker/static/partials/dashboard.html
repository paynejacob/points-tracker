Welcome to the poor floor stamp management system!

ATTENTION USERS!

This is a pre-alpha release

  • There is a known bug that allows multiple files to be played simultaneously please wait for tracks to finish playing before clicking play on another track
  • Please take your time naming files and tagging them, this makes it easier to search and easier to read the track names
  • The supported filetypes are mp3, mp4 and wav
  • Please report issues here
  • If you would like to contribute to this project contact Jacob Payne
  • Audio files have a maximum length of 15 seconds, if you upload a file longer than 15 seconds only the first 15 seconds will be saved
